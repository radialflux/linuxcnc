Linuxcnc

LinuxCNC is a software system for computer control of machines such as milling machines, lathes, plasma cutters, cutting machines, robots, hexapods, etc.

Quickstart:

From the top level directory, switch to the source directory:

cd src

In the source directory, build LinuxCNC:

[for rtai] ./configure [or, for PREEMPT-rt or vanilla kernels:] ./configure --with-realtime=uspace

make clean make sudo make setuid

to run the software go back to the top level directory, and issue:

. scripts/rip-environment linuxcnc
